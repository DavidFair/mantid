.. _func-Polynomial:

==========
Polynomial
==========

.. index:: Polynomial

Description
-----------

.. attributes::

   n;Int;0;The order of the highest term in the polynomial

.. properties::

For every m, :math:`0 < m <= n`, an additional parameter Am is generated,
representing the coefficient of the :math:`x^m` term, with default 0.
